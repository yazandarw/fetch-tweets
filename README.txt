Hinweis:

Das Projekt wurde unter Verwendung der Node Js-Technologie entwickelt. Daher muss "Node Js" und "MongoDB" installiert werden. Zusätzlich müsste auch einige Pakete wie "express" durch npm installiert werden. Außerdem könnte es sein, dass einige Module in "Package.json" implementiert werden müssen, wie z.B "mongoose", "jsonwebtoke", "twit" und usw, damit das Projekt gut funktioniert.

Außerdem mussen die Access-Token, Access-Token-Secret und die MongoDB-URL eingegeben werden.
